Comment créer les dialogues sur lesaventuresdesteven.exe

Sur Godot dans le système de fichiers, faites clic droit sur le dossier "dialogues" > Créer nouveau > Resource
Ensuite choisir "D_Timeline"

Un "D_Timeline" consiste simplement en une liste de D_Object
Les D_Object peuvent être :
	Un perso qui parle (D_Message)
	Un temps de pause (D_Pause)
	Un événement quelconque (D_Event)

D_Message :
	Nom Perso : le nom du personnage qui parle
		Si c'est une voix off, ne rien mettre
	
	Mess : le message a écrire
	Si vous voulez mettre des effets en particulier (pour l'instant les effets sont pas implémentés), il existe deux balises :
		[pause]<temps de pause en secondes>[\pause] : Créer une pause dans le message.
		Exemple : [pause]0.5[\pause]
	
		[glitch]blablabla[\glitch] : Glitch le message
	
	Liste choix : une liste de choix après qu'un message soit complètement écrit
		Si pas de choix -> laisser vide
		Sinon, créer des choix qui mène vers d'autres timelines
			Libelle Choix : nom du choix fait
			Goto : La timeline que vous pouvez créer soit directement dessus, soit en créant une nouvelle resource comme expliqué plus haut et le référencer dans le choix
			Variable Condition : Le nom d'une variable a tester sur le joueur qui conditionne l'affichage du choix
		ATTENTION : Si la liste de choix du message n'est pas vide, la timeline actuel ne continuera pas sur les messages suivants mais sur une autre timeline
	
	Event At Start : Un nom d'événement qui se lance avant le début de l'écriture du message

D_Pause:
	Temps : Le temps de pause en secondes
	Affiche Dialogue : Est-ce que pendant la pause, la fenêtre de dialogue est affiché ?

D_Event:
	Nom Event : Le nom de l'event à lancer
	Parallele : Doit-on attendre qu'un signal "fin_event" soit lancé (false) ou est-ce qu'on passe en même temps de l'event sur le message suivant (true)
	
Tous les noms d'event doit être notifié afin que je puisse coder son comportement
Si vous voulez tester le dialogue, il faut :
	- Importer le dialogue : Dans Projet > Paramètres de projet > Généraux > GestionDialogue, puis en insérant dans la variable liste_timeline un nom et le chemin vers le dialogue
		Exemple : "nom_dialogue" : preload("res://dialogues/dialogue.tres")
		PAS OUBLIER LA VIRGULE À LA FIN DE TOUTES LES ENTREES
	- Lancer un dialogue test : Dans Projet > Paramètres de projet > Généraux > Globals, puis en changeant le nom de la timeline à lancer
		--> GestionDialogue.lancer_timeline(<nom du dialogue défini dans GestionDialogue>)
